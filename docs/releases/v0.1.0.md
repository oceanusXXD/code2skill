# code2skill v0.1.0

## Summary

This release establishes the initial public presentation for `code2skill`.

The focus is not on changing the core pipeline, but on making the package easier to discover, understand, and evaluate on GitHub and PyPI.

## Highlights

- Reworked the main README into a clearer English landing page
- Added a dedicated Chinese document: `README.zh-CN.md`
- Improved the PyPI short description and search keywords
- Kept the validated `0.1.0` feature scope intact

## What users can understand more quickly

- what `code2skill` generates
- how `scan`, `estimate`, `ci`, and `adapt` fit together
- which AI tools the generated outputs target
- how to configure OpenAI, Claude, and Qwen
- what the current Python-only scope is
